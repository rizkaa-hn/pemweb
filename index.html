HTML




  MENGENAL HTML


HTML merupakan singkatan dari Hypertext Markup Language, yaitu bahasa standar web yang dikelola penggunannya
oleh W3C (World Wide Web Consortium) berupa tag-tag yang menyusun setiap elemen dari website. HTML berperan
sebagai penyusun struktur halaman website yang menempatkan setiap elemen website sesuai layanan yang diinginkan.

HTML biasanya disimpan dalam sebuah file berekstensi .html. Untuk mengetikkan skrip HTML, kita dapat menggunakan
text editor seperti Notepad sebagai bentuk paling sederhana atau text editor khusus yang dapat mengenali setiap unsur skrip
HTML dan menampilkannya dengan warna yang berbeda sehingga mudah dibaca, seperti Notepad++, Sublime Text,
Visual Studio Code, dan masih banyak lagi aplikasi yang sejenis.




  UNSUR HTML


HTML terdiri atas unsur-unsur yang membentuk struktur skrip HTML, yaitu tag, atribut, dan elemen.

Tag
Tag adalah simbol khusus (markup) berupa dua karakter "<" dan ">" yang mengapit suatu teks sebagai nama tag. Sebagian
besar tag memiliki tag pembuka dan tag penutup yang membatasi pengaruh dari tag tersebut.
Contohnya adalah: <body> sebagai pembuka tag dan </body> sebagai penutup tag

Atribut
Atribut adalah properti yang mengatur bagaimana elemen dari suatu tag akan ditampilkan. Atribut ditulis di dalam simbol
tag setelah nama tag dengan dipisahkan oleh spasi. Atribut ada yang memiliki nilai dan ada yang tidak memiliki nilai.
Nilai dalam suatu atribut ditulis di dalam tanda petik ganda ("..."), dipisahkan dengan simpol sama dengan (=) dari nama
atribut, misalnya <p aling="center">.

Element
Element adalah bagian dari skrip HTML yang terdiri atas tag pembuka, isi element, dan tag penutup. Jika sebuah element
ditampilkan pada browser, hanya isi element yang tampil di browser.
Sebagai contoh: <p>Pemograman Web</p>
Jika ditampilan, hanya tulisan "Pemograman Web" yang akan tampil.
